<h2>Obrigado pela visita!</h2><br>
<h4>Este projeto consiste no desenvolvimento de um conversor de moedas, onde o usuário seleciona a moeda a ser convertida e o valor e a aplicação realiza o calculo.<br>
Para este projeto foram utilizadas as tecnologias HTML, CSS e JavaScript.<br>
Também foram aplicadas técnicas de responsividade com o objetivo de aplicar o projeto a diferentes dispositivos.<br><br>
Confira o resultado do projeto <a href="https://luckasmatos-currencyconverter.netlify.app/" target="_blank">AQUI</a>.</h4>
